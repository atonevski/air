# Air
This is small application which collects air pollution
measurements on the teritory of Macedonia on the following
stations:
* Центар
* Карпош
* Лисиче
* Гази Баба
* Ректорат
* Миладиновци
* Мршевци
* Битола 1
* Битола 2
* Кичево
* Лазарополе
* Тетово
* Велес 1
* Велес 2
* Кочани
* Кавадарци
* Куманово
